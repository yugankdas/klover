# Klover VS Code Extension

Syntax highlighting and language support for the **Klover** UI language.

## Features

- **Syntax Highlighting**: Full support for .kv files.
- **Theme Support**: Highlighting for dark and light themes.
- **Metadata Support**: Highlighting for title, icon, and favicon keywords.
- **Responsive Logic**: Support for column, row, if, and repeat.
- **Typography Variants**: Highlighting for h1-h4, heading, and subheading.

## Installation

1. Search for **Klover** in the VS Code Marketplace.
2. OR install the .vsix file manually.

## File Icons

To enable Klover file icons:

1. Open Command Palette (Ctrl + Shift + P).
2. Search: File Icon Theme.
3. Select: Klover Icons.

## Requirements

Requires Klover CLI for building and running files.
